# Projeto Vaga Estágio

Este repositório contém um projeto desenvolvido para um processo seletivo de estágio. O objetivo principal foi demonstrar habilidades em desenvolvimento web, seguindo boas práticas e aplicando conhecimentos adquiridos em estudos e experiências acadêmicas.

## 🛠 Tecnologias Utilizadas
- HTML
- CSS
- JavaScript
- Git e GitHub

## 📌 Objetivo do Projeto
O projeto foi criado com fins de estudos e para demonstrar competências técnicas relacionadas ao desenvolvimento web. Ele inclui funcionalidades como manipulação de DOM, estilização responsiva e interatividade com JavaScript.

## 🚀 Funcionalidades Implementadas
- Estruturação semântica com HTML
- Estilização moderna e responsiva com CSS
- Interatividade com JavaScript
- Uso de controle de versão com Git

## 📋 Requisitos para Execução
Para rodar o projeto localmente, siga os passos:
1. Clone o repositório:
   ```bash
   git clone https://github.com/micaelleitee/projeto-vagaestagio.git
   ```
2. Acesse a pasta do projeto:
   ```bash
   cd projeto-vagaestagio
   ```
3. Abra o arquivo `index.html` no navegador.

## 🎯 Aprendizados e Experiência
Durante o desenvolvimento deste projeto, foram exercitadas as seguintes competências:
- Estruturação e estilização de páginas web
- Escrita e revisão de código utilizando boas práticas
- Versionamento com Git e GitHub
- Testes de funcionalidade e responsividade
